# mini-projeto-python---Franciso-Lemos---Vladimir-Gustavo

Projeto 1 – Cadastro de Produtos

Sistema simples de controle de estoque feito em Python.
Permite cadastrar, listar, buscar, atualizar e excluir produtos.

Funções principais:
Cadastro com código único (usa set para evitar repetição)
Armazena produtos em uma lista
Categorias definidas em uma tupla
Usa while e for
Mostra mensagens amigáveis e valida erros

Projeto 2 – Controle de Alunos e Notas

Sistema para cadastrar alunos, registrar notas e calcular médias.
Funções principais:
Cadastro de alunos
Registro de notas
Cálculo de média
Relatórios de alunos, médias e situação (aprovado/reprovado)
